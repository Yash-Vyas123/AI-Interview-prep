# AI Interview Prep (MERN)


## Project Summary

Built a MERN-based AI Interview Prep platform with secure JWT authentication, quiz engine, mock AI feedback, and user progress tracking suitable as a portfolio and college submission project.

## Overview
AI-powered interview practice app with quizzes, scoring, feedback, and progress tracking.

## Features
- JWT auth (login/register)
- Admin question management (backend)
- Quiz with scoring and feedback
- User history of attempts

## Tech Stack
- MongoDB, Express, React, Node.js
- JWT, bcrypt
- (Planned) OpenAI / Gemini API for real feedback

## How to run

### Backend
1. `cd backend`
2. `npm install`
3. `npm start`
Backend runs at: `http://localhost:5000`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm start`
Frontend runs at: `http://localhost:3000`
